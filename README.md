## What It Is

Anti-AI Cyber Hack is a browser game made entirely with HTML, CSS, and JavaScript, intended to run offline. Its design blends clicker-style progression with a cyber-themed loop, delivering a lightweight experience that does not depend on third-party libraries.

Built for HTML5 environments, it is also comfortable on mobile browsers. The package includes RU/EN localization, three difficulty tiers, and local language persistence so the chosen interface language remains available across sessions.
